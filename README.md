# desikau.com

## 環境構築

```
docker compose up -d --build

```

## ローカル環境のセットアップ
hostsファイルに追加 (/etc/hosts):
```
127.0.0.1 desikau.local
127.0.0.1 admin.desikau.local
127.0.0.1 api.desikau.local
```

## アクセスURL:
- ユーザーサイト: http://desikau.local:8080
- 管理画面: http://admin.desikau.local:8080
- API: http://api.desikau.local:8080
